# gpt3-korean

gpt3 with papago

## Installation (macOS)

```shell
git clone https://github.com/junhsss/gpt3-korean.git
cd gpt3-korean
virtualenv venv
. venv/bin/activate
pip install -e .
```

## Usage

- 한국어 : 파파고 -> gpt3 -> 파파고

```shell
ask --question "기계와 두뇌는 무엇이 다른가?"
```

- 영어 : gpt3 -> 파파고

```shell
ask --question "Brains, Minds and Machines. What are the differences?"
```
