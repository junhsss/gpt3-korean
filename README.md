# gpt3-korean

gpt3 with papago

## Installation (macOS)

```shell
git clone https://github.com/junhsss/gpt3-korean.git
cd gpt3-korean
virtualenv venv
. venv/bin/activate
pip install -e .
```

## Usage

```shell
ask --question "기계와 두뇌는 무엇이 다른가?"
```

```shell
ask --question "What is the mind? what's its relationship with the machine and brain"
```

## Papago

.env 파일에 네이버 API 키 입력하세용.
